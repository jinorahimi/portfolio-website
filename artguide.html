artguide Redesign

Role

UX/UI Designer

Tools

Sketch, InVision, Whimsical, Principle

Context

artguide users need a fully searchable database that provides easy access to the best events for contemporary art and enables them to access a handy art guide for any city. The current tool for doing so creates a frustrating experience for art enthusiast.

It is difficult for users to filter and sort their search and keep track of exhibitions they would like to see. There are multiple menus and the product relies heavily on textual information for navigation, creating a counterintuitive user experience. The solution has therefore had a negative impact on the number of people using the existing responsive web product.

Objective

My goal with this project was to redesign artguide searchable database of current art events to provide art enthusiasts with a more usable experience finding contemporary art exhibitions. My focus was mainly in making the search feature more intuitive and simple. In addition, I created visual icons for my design to help users quickly and easily identify the features they need.

Who

Contemporary art enthusiasts as well as professionals in the art field that are looking for current art exhibitions in any city they may visit.

What

The responsive web application artguide, which is an exclusive feature of artforum.com. The product has a searchable database of exhibitions, cities, and venues. Users can browse the database, select a venue or event to find all the available information, share an event listing with friends, and save an event to their personalized artguide.

When

The responsive web product can be used any time to access a handy art guide at user’s fingertips.

Where

Art enthusiasts can use this product at home or on the go. Users can search for art events anywhere.

Why

So more people can enjoy and appreciate Contemporary art and learn about artistic events happening around them.

Feature Requirements

User registration and login process to save and create a personalized art guide.

Search and sort feature for finding current events including exhibitions, openings and closings, and art fairs in any city.

Access comprehensive information about any event including event dates, images, press release, venue’s address, and opening hours.

Save an event listing to user's artguide.

Map feature to view the location of venues in order to plan a day of gallery hopping in advance.

Ability to contact a venue and book appointments online (if required by the venue).

Local editor’s recommendations feature for each city.

Share an exhibition listing with friends. Send by email, text message, or social media.

User Stories

As an artist, I want to be able to stay on top of upcoming exhibition openings in order to spend an evening with my peers and expand my professional network.

As an art collector, I want to be able to search for exhibitions in my city to find the next addition to my collection.

As an art enthusiast, I want to create a personalized art guide when I travel to a new city and view my guide on a map in order to explore the local art scene.

As an art lover, I want to be able to save exhibitions that I visit, so that I can easily revisit them and look up the participating artists to learn more about their work.

User Flow

Moodboard

I picked the first moodboard for my redesign project. Exhibition spaces often have minimal design with white walls and blue-gray concrete floors to highlight the art on view. This color palette is a great option for my application because it mostly uses neutral colors familiar to art professionals. I included black to increase contrast for legibility and accessibility reasons, while keeping the palette simple. Black is modern and trendy in the art world. White evokes feelings of simplicity and communicates a calm and professional atmosphere. Additionally, blue-gray communicates serenity and relaxation.

Redesign

Using icons and animation helps users quickly identify the features they need, while keeping the interface simple.

In the existing solution, features such as adding an event to artguide, sharing an event, as well as the complete press release are presented as text. It is difficult to identify these features as they are placed somewhere in the middle of the page. By moving these features to the top of the page and displaying them in the form of icons, users can easily navigate the page.

The addition of animation makes the screen easier to understand. This slight change in size and color can quickly tell users that they’ve successfully added an event to their artguide. By organizing the information in sections, we allow users to know where to direct their attention.

In the existing solution, user needs to scroll through a list of hundreds of cities in order to view events in a particular city. In my redesign, I included an autocomplete search feature that makes searching much more usable than a typical text field and submit button. This allows users to quickly autocomplete their search, saving time and providing insight into what search options are available. My goal was to offer a minimal and clean style that is consistent with my moodboard.

In the existing design, there are two different menus with overlapping features. One of the menus is located in the middle of the screen that confuses users.

In my redesign, I consolidated the two menus into one to avoid confusion. I added a separate user profile icon that takes user to their personalized artguide and account details.

Currently, filtering and sorting are both accessible through a button that is labeled “category”. This has a negative impact on the usability of the product. In my design, I created two separate buttons with visual icons that allows users to easily and quickly organize their search results.

The map feature is only available in the desktop version of the existing solution. This is a highly useful feature that lets users to view the location of venues when planning a day of gallery hopping in advance. Users find it helpful to be able to toggle between the map and list view.
